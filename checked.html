about the app

Checked is a to-do app that allows users to manage and organize their tasks. Users can mark each task as low, medium, or high priority, add notes, and set deadlines and/or reminders. The to-do list can be sorted by date created, deadline, or priority, or filtered by past due, completed, and more. The app also features 3 different size widgets that display pending tasks.

The UI of the app was inspired by Herdetya Priambodo's design on dribbble.com.
