Pyrls Vision

This is an app that I developed as an intern at Cosmas Health. and it focuses on optical character recognition (OCR). The app allows users to find drug information and check for interactions from a list of drugs. Users can search for drugs by manually typing the drug's name or by using their camera to take a photo of text. A simplified, scaled-down version of this app can be found on my Github .

what i learned

  • Perform multiple network requests using Combine
  • ML Kit Vision's text recognition API
  • Better error handling with Result type and enums
  • Create an onboarding screen using Tabview and @AppStorage

see it in action


Basketball Stats Checker

A simple, iOS app that allows you to find player stats for the current NBA season. My goals for this app were to get familiar with Combine and unit testing, as well as practice making network requests and retrieving data from a web API. This app and a more detailed explanation of how I created it can be found on my Github .

what i learned

  • Perform network requests and field validation with Combine
  • Write more testable code using protocols and mock services
  • Write and run tests with the XCTest Framework
